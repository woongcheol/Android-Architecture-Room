## 본론
### 1. Android Room이란?

SQLite 개체 매핑 라이브러리이다. 안드로이드에서 권장하는 라이브러리로, 이에 대한 장점은 후술한다.
Jetpack에서 제공하는 라이브러리 중 하나로, 앱 아키텍쳐 권장사항에 해당하는 AAC(Android Architecture Components)에서 제공한다.
